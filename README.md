# UniAppInterlocution
基于springboot和uniapp的智能问答论坛社区小程序

演示视频链接:https://live.csdn.net/v/272298

项目实现功能

微信用户登录后可以进行提问和解答，问题可根据热度、时间和距离进行排行

智能机器人在线解答问题功能，使用第三方API

个人中心可查看回答、提问、收藏

使用的技术

后端：SpringBoot框架，MySql数据库

前端：UniApp框架（可实现跨平台开发包括H5、小程序、安卓或苹果APP）
